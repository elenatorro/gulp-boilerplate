#Gulp Boilerplate

Basic Gulp application with Javascript and Sass

##Instalation

* Requisites
	* npm installed
	* Gulp installed 
* Clone de repo

```
	$ git clone https://github.com/elenatorro/gulp-boilerplate.git
	$ cd gulp-boilerplate
```
* Install modules

```
	$ sudo npm install
```

* Run the server

```
	$ gulp
```

##Scaffolding
